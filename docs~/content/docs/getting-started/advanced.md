## 処理順序

ModEmoでは、Hierarchyで上に配置されているオブジェクトが優先的に処理されます。


## まばたきを追加する

[ルートコンポーネント](../../components/mod-emo-component) の直下に配置された表情は、まばたき用の表情として登録されます。  
また、まばたき用表情を作るためのコンポーネントとして [ModEmo Blink Expression](../../components/expression/blink/) が用意されています。
